# 安全政策

## 报告安全问题

请不要在公开 Issue 中提交密码、令牌、手机号、VIN、位置、订单号、完整日志、Home Assistant `.storage` 目录、数据库或备份。

发现安全问题时，请通过 GitHub 的私密漏洞报告功能联系维护者：进入仓库的 **Security → Report a vulnerability**。报告中只提供复现所需的最小信息，并先替换所有账户、车辆、位置和充电记录标识。

## 数据边界

本集成不向维护者服务器发送遥测。账号密码由 Home Assistant 配置条目在本地保存；车辆状态、VIN、位置和充电数据可能进入实体注册表、Recorder 数据库、日志与备份。它们是否会被同步到其他服务，取决于用户自己的 Home Assistant 配置。

代码会对已知敏感字段进行日志脱敏，但这不能替代人工检查。分享诊断信息前，请确认其中不含个人数据。
