# Volvo On Call CN v3.0.2

本版本基于原项目做最小功能补充，重点是混动车型电类实体、TM/TA 行程信息、动力类型过滤、驻车空调开关和轮询稳定性。

## 主要变化

- 混动车型新增动力电池电量、纯电续航、充电状态、充电枪连接状态、预计充满时间和充电功率。
- 新增 TM/TA 行程信息：里程、平均速度、平均油耗；TM 额外支持平均电耗。
- 新增 `sensor.{vin}_full_charge_electric_range`，在电量首次达到 `100%` 时采样当次纯电续航，支持 Home Assistant 长期统计。
- 新增轻混/纯油与混动动力类型过滤，轻混/纯油不创建或轮询电类实体。
- 恢复驻车空调开关：仅开启/关闭空调，不设置运行时长。

## 稳定性和安全修复

- 定时刷新和操作后手动刷新统一受协调器扫描间隔节流，最小扫描间隔为 30 秒。
- 数字沃尔沃 HTTP 请求失败日志会脱敏 refresh token、Bearer token 和 X-Token。

## 验证

- Python 单元测试覆盖满电采样、动力类型过滤、TM/TA 元数据和现有协调器行为。
- Node 前端测试覆盖卡片注册、实体推导、统计建议、危险操作确认和充电状态判断。
